## Installation
Using Gradle:
```groovy
testImplementation 'io.github.mfvanek:pg-index-health-test-starter:0.4.0'
```

Using Maven:
```xml
<dependency>
  <groupId>io.github.mfvanek</groupId>
  <artifactId>pg-index-health-test-starter</artifactId>
  <version>0.4.0</version>
  <scope>test</scope>
</dependency>
```
